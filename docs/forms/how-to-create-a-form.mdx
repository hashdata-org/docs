# Como criar um formulário
Os formulários são a principal ferramenta para a realização da coleta de dados.
A criação correta de um formulário influencia diretamente no sucesso da análise dos dados coletados, pois garante as
informações relevantes sejam capturadas de maneira organizada.

A definição adequada das perguntas e dos tipos de resposta, como campos de números, de texto, data e hora ou múltipla escolha,
é fundamental para que os dados possam ser facilmente analisados e convertidos em conhecimentos para as tomadas de decisão.

Para criar ou editar um formulário, gerenciando suas funcionalidades, clique em <img style={{width: '2.5rem'}} src={require('../start-here/images/main-menu/forms.webp').default} alt="Formulário"
className="no-zoom image-between-texts" />, disponível no [Menu Principal](/start-here/main-page#main-menu).

## 1. Defina a Área de Trabalho {#set-the-desktop}
Certifique-se que a área de trabalho ativa é aquela na qual você deseja criar o formulário. Formulários irão pertencer
à área de trabalho que estiver ativa no momento de sua criação.

:::tip[Mudar Área de Trabalho]
[Clique aqui](/start-here/main-page#change-desktop) e veja mais detalhes de como mudar a área de trabalho ativa.
:::

## 2. Organize os Formulários em Pastas {#organize-the-forms-into-folders}
As **pastas** são uma forma de organizar os formulários em uma estrutura hierárquica, permitindo agrupá-los
conforme suas características, finalidades, temas, projetos, datas ou departamentos. Essa organização facilita a
navegação, o gerenciamento e o controle de acesso, especialmente quando se lida com um grande volume de formulários.

:::danger Importante!
A organização dos formulários em **pastas** juntamente com a organização dos **colaboradores** e das **contas de
dispositivo** em **grupos (unidades organizacionais)** são a melhor forma de se gerenciar o controle de acesso aos
formulários de uma área de trabalho.

Com essa abordagem, basta conceder aos grupos os acessos às devidas pastas. Desta forma, quando forem incluídos
colaboradores nos grupos e formulários nas pastas, as configurações de acesso serão automaticamente aplicadas,
facilitando a gestão de acessos aos formulários.

[**Saiba mais**](/forms/setting-up-forms-access) como **configurar o acesso aos formulários**.

[**Saiba mais**](/collaborators/#groups) como cadastrar **grupos**.

[**Saiba mais**](/collaborators/#invite-collaborators) como **convidar colaboradores**.
:::

Quando um formulário for criado, será mostrada opção para ser definida qual a pasta ou subpasta que o formulário
deverá ser criado.

### Gerenciamento de Pastas {#folders-management}
1. Para incluir pasta de primeiro nível, clique em **Nova pasta** <img style={{width: '2rem'}}
src={require('../images/plus_button.webp').default} alt="Nova pasta"
className="no-zoom image-between-texts" />, escolha o nome da pasta e confirme;
2. À direita de cada pasta, clique no botão **Mais opções...** <img style={{width: '1.7rem'}}
src={require('../images/ellipsis_vertical.webp').default} alt="Mais opções..."
className="no-zoom image-between-texts" />:
 - **Nova subpasta**: cria uma subpasta;
 - **Alterar**: altera o nome da pasta;
 - **Grupos autorizados**: define quais **grupos** terão acesso à pasta;
 - **Excluir**: exclui a pasta.
3. O botão **Grupos autorizados** <img style={{width: '1.7rem'}}
src={require('../images/authorized_groups.webp').default} alt="Grupos autorizados"
className="no-zoom image-between-texts" /> permite gerir quais são grupos que terão acesso à pasta.
Ao clicar nesse botão, são mostradas, na porção direita da tela, todos **grupos** disponíveis. Selecione os grupos
os quais devem ter acesso à pasta.

:::warning Heranças de acesso
Subgrupos não herdam autorização de acesso do grupo pai (ascendente), ou seja, ao se autorizar um grupo para ter acesso
a uma determinada pasta, os subgrupos dele não herdam o acesso à pasta.
Logo, a determinação de quais pastas cada grupo pode acessar deve ser realizada individualmente (grupo a grupo).
:::

## 3. Crie um Formulário {#creating-a-form}

1. Clique em <img style={{width: '7rem'}}  src={require('./images/how-to-create-a-form/click_to_new_form.webp').default} alt="Novo formulário"
className="no-zoom image-between-texts" />;
2. Defina o nome do formulário;
3. Selecione a pasta ou subpasta de destino;
4. Clique em "Criar" ou utilize a tecla de atalho "Ctrl+S.

:::tip[Gerenciamento de pastas e criação de formulários]
Veja a figura no início desta paǵina. Ela mostra as principais funcionalidades da tela **Formulários**, incluíndo o
**gerenciamento de pastas** e a **criação de um formulário**.
:::

## 4. Adicione Elementos ao Formulário {#add-elements-to-the-form}
Após a criação do formulário, será aberta a tela de edição da estrutura e das funcionalidades do formulário criado.
Na barra lateral esquerda dessa tela, estão disponíveis os **elementos**.

Os **elementos** são os componentes que constituem a **estrutura** do formulário, permitindo a coleta dos dados.
Em outros termos, os elementos são os tipos de perguntas disponíveis para serem inseridos em um formulário.

[**Saiba mais**](/forms/elements/what-is-elements) como adicionar e configurar elementos no formulário.
