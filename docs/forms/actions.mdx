# Ações

### Simular respostas
A funcionalidade de **Simular respostas** gera respostas aleatórias para o formulário antes de publicá-lo.
Dessa forma, é possível realizar análises gráficas, aplicar filtros e certificar-se de que os dados serão coletados no formato desejado.

:::info Fique tranquilo(a)
As respostas geradas serão descartadas automaticamente ao publicar o formulário.
:::

### Copiar elementos de outro formulário
Com a funcionalidade **Copiar elementos de outro formulário**, é possível reaproveitar perguntas, seções e configurações
já criadas em outros formulários. Isso otimiza o tempo de configuração, especialmente ao criar formulários com conteúdos
semelhantes. Após importar os elementos, você pode personalizá-los conforme necessário para atender às novas demandas.

### Configurar coletas pendentes
A opção **Configurar coletas pendentes** é usada para gerenciar formulários que ainda não foram completamente respondidos
ou enviados.
Quando uma coleta é iniciada mas não é finalizada, ela fica salva como 'Pendente' e pode ser concluída em momento posterior.
Contudo, é necessário definir como identificar as coletas pendentes para diferenciá-las umas das outras. Selecione as perguntas
cujas respostas serão usadas para identificar as coletas pendentes.

### Usuários autorizados
A funcionalidade **Usuários autorizados** define quais colaboradores podem acessar o formulário. Porém, as ações que cada
colaborador pode realizar são definidas na tela de perfil/permissões de cada colaborador.

:::warning Recomendação
Evite configurar o acesso por colaborador ou por formulário individualmente, ao invés disso, junte os colaboradores em
grupos (unidades organizacionais) e os formulários em pastas; depois autorize os grupos a acessar as pastas devidas.
Assim, quando houver novos colaboradores ou novos formulários, as configurações de acesso serão aplicadas automaticamente.
:::

### Clonar
A funcionalidade **Clonar** permite criar uma cópia exata de um formulário existente. Isso é útil quando você deseja
criar um novo formulário com a mesma estrutura e configurações de outro já existente, economizando tempo de configuração.
Após clonar, é possível personalizar o formulário clonado sem alterar o original.

### Compartilhar
A funcionalidade **Compartilhar** possibilita enviar o formulário para outras pessoas por meio de um link ou compartilhamento
direto. Dependendo das configurações, você pode tornar o formulário público (acessível a qualquer pessoa com o link) ou
restringi-lo a usuários autorizados, garantindo que apenas os destinatários desejados tenham acesso.

### Visualizar logs do formulário
A funcionalidade **Visualizar logs do formulário** permite acompanhar todas as alterações realizadas no formulário desde sua
criação. Os logs incluem informações como mudanças de configurações, inclusão ou exclusão de perguntas e atualizações feitas por
diferentes usuários, proporcionando transparência e controle.

### Visualizar logs de respostas
A opção **Visualizar logs de respostas** registra todas as interações realizadas nas respostas coletadas. É possível verificar
quem respondeu, editou ou excluiu uma resposta, bem como acompanhar alterações feitas em campos específicos. Isso é essencial para
auditoria e rastreamento de dados.

### Excluir todas as respostas
A funcionalidade **Excluir todas as respostas** permite apagar de forma definitiva todos os dados coletados em um formulário.
Essa ação é irreversível e deve ser utilizada com cautela, especialmente em casos de formulários já vinculados a relatórios
ou exportações.

### Excluir formulário
A opção **Excluir formulário** remove permanentemente o formulário selecionado e todas as suas configurações. Antes de
realizar essa ação, é importante verificar se o formulário ainda é necessário, pois a exclusão é irreversível e todos os
dados e respostas serão perdidos.
