# Ações Condicionais


As ações condicionais servem para deixar o seu formulário inteligente e são aplicadas automaticamente nos elementos do
formulário quando determinadas condições acontecem.

A estrutura lógica dessas ações são do tipo **SE/ENTÃO**: **SE** (condição acontecer) **ENTÃO** (faça a ação).

Por exemplo, pode-se utilizar as **ações condicionais** para ocultar ou exibir elementos do formulário quando determinada
condição acontecer, tal como:
* **SE** a opção X da questão Y for selecionada, **ENTÃO** oculte as questões A, B, C e exiba as questões D, E, F.

:::warning Atenção
As Ações condicionais são processadas **SEMPRE** que uma resposta do formulário é alterada. Ou seja, as
modificações realizadas em um formulário serão aplicadas a todas as respostas prévias.
:::
