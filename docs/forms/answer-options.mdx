# Opções de Respostas

Para os elementos do tipo **Seleção única** e **Múltipla seleção** é necessário configurar os itens de respostas que
serão apresentados no formulário para a coleta de dados.

A funcionalidade **Opções de Resposta** permite customizar as opções que serão apresentadas para o usuário, ajustar
os atributos de cada opção, a ordem em que aparecem, incluir etiquetas e valores aos itens.

## Para incluir opções de resposta
1. Clique em <img style={{width: '8%'}} src={useBaseUrl('img/forms/answer-options/add_item.webp')}
alt="Incluir item" className="no-zoom image-between-texts"/>;
2. Preencha o campo texto com a descrição da opção;
3. Preencha os campos *Etiqueta (identificador)* e *Valor*, caso estejam habilitados.
4. Para mudar a ordem dos itens, clique em <img style={{width: '2%'}} src={useBaseUrl('img/forms/answer-options/move.webp')}
alt="Mover item" className="no-zoom image-between-texts"/> (à esquerda da opção), segure e arraste para a posição desejada.
5. Clique em <img style={{width: '8%'}} src={useBaseUrl('img/forms/save.webp')}
alt="Salvar" className="no-zoom image-between-texts"/>, para guardar as configurações.

## Para importar opções de resposta
1. Clique em <img style={{width: '10%'}} src={useBaseUrl('img/forms/answer-options/import.webp')}
alt="Importar" className="no-zoom image-between-texts"/>. Será aberta uma caixa de diálogo com as opções de importação;
2. Defina os campos que farão parte da importação das opções: *Texto* (obrigatório sempre), *Etiqueta*, *Valor*, *Permitir comentários* e *Permitir fotos*;
2. Escolha o delimitador dos campos. Veja as opções no menu suspenso;
4. Digite ou cole as opções que serão importadas;
5. Confira os valores e clique em <img style={{width: '8%'}} src={useBaseUrl('img/forms/answer-options/import2.webp')}
alt="Importar" className="no-zoom image-between-texts"/>, para concluir.
