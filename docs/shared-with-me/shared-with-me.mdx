# Compartilhados Comigo

A funcionalidade **Compartilhados comigo** destaca-se pela praticidade e eficiência na colaboração entre equipes. Essa
funcionalidade permite gerenciar os formulários criados por outros colaboradores e que foram compartilhados diretamente
com você, promovendo um ambiente de trabalho integrado e colaborativo.

A partir de um formulário compartilhado, é possível utilizá-lo como modelo para a criação de um novo formulário. Ou seja,
utilizar a mesma estrutura e configurações já existentes, economizando tempo de configuração. O novo formulário poderá ser
personalizado, sem alterar o formulário compartilhado.

:::warning Atenção
Quando um formulário é compartilhado com você, apenas sua estrutura fica disponível. Os dados coletados não são
compartilhados.

Saiba mais sobre [**Compartilhar**](/forms/actions#sharing)
:::
#### Principais benefícios {#main-benefits}
- **Acesso centralizado**: todos os formulários que foram compartilhados ficam organizados nesta tela, facilitando a
localização e o gerenciamento;
- **Colaboração eficiente**: equipes podem compartilhar formulários já prontos, que servem como modelos para a criação
de novos formulários, reduzindo o tempo de criação e, consequentemente, melhorando a produtividade do time.



Para acessar a funcionalidade, clique em **Compartilhados comigo** <img style={{width: '2.5rem'}}
