Como criar um formulário

Os formulários são a principal ferramenta para a realização da coleta de dados. A criação correta de um formulário influencia diretamente no sucesso da análise dos dados coletados, pois garante as informações relevantes sejam capturadas de maneira organizada.

A definição adequada das perguntas e dos tipos de resposta, como campos de números, de texto, data e hora ou múltipla escolha, é fundamental para que os dados possam ser facilmente analisados e convertidos em conhecimentos para as tomadas de decisão.

Para criar ou editar um formulário, gerenciando suas funcionalidades, clique em , disponível no Menu Principal.

1. Defina a Área de Trabalho​

Certifique-se que a área de trabalho ativa é aquela na qual você deseja criar o formulário. Formulários irão pertencer à área de trabalho que estiver ativa no momento de sua criação.

Mudar Área de Trabalho

Clique aqui e veja mais detalhes de como mudar a área de trabalho ativa.

2. Organize os Formulários em Pastas​

Organizar formulários em pastas permite uma gestão mais eficiente, especialmente quando se lida com um grande volume de formulários.

As pastas e subpastas ajudam a categorizar os formulários por temas, projetos, datas ou departamentos, garantindo que cada um esteja no local correto. Além disso, essa estruturação possibilita uma visualização rápida, facilitando a localização de formulários antigos ou específicos, melhorando a colaboração entre equipes.

Quando um formulário for criado, será mostrada opção para que seja definida qual a pasta ou ou subpasta que o formulário deverá ser criado.

Gerenciamento de Pastas​

No gerenciamento de pastas, você pode:

  • Incluir uma nova pasta;
  • Incluir uma subpasta;
  • Alterar o nome de uma pasta;
  • Definir os grupos de colaboradores que terão acesso à uma determinada pasta, e;
  • Excluir uma pasta.

3. Crie um Formulário​

  1. Clique em ;
  2. Defina o nome do formulário;
  3. Selecione a pasta ou subpasta de destino;
  4. Clique em "Criar" ou utilize a tecla de atalho "Ctrl+S.

4. Adicione Elementos ao Formulário​

Após a criação do formulário, será aberta a tela de edição da estrutura e das funcionalidades do formulário criado. Na barra lateral esquerda dessa tela, estão disponíveis os elementos.

Os elementos são os componentes que constituem a estrutura do formulário, permitindo a coleta dos dados. Em outros termos, os elementos são os tipos de perguntas disponíveis para serem inseridos em um formulário.

Saiba mais como adicionar e configurar elementos no formulário.

5. Configure o Formulário​

As configurações do formulário permitem personalizar a maneira como os formulários são apresentados e processados.
