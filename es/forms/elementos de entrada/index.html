Elementos (entrada de datos)

Los elementos son los tipos de preguntas disponibles para la creación de formularios y se dividen en dos grandes grupos, 1) Entrada de datos y 2) Visualización.

Los elementos de entrada de datos se subdividen en: texto, números, selección única, selección múltiple, fecha y hora, evaluación y **otros **.

Se accede a ellos desde la pantalla de edición del formulario:

Para información detallada accede a la sección formularios.

Elementos de tipo de texto​

Los elementos de tipo texto son aquellos en los que la entrada de datos es una cadena (texto). En este tipo de ingreso de datos se generan los siguientes informes: extracto de colección y tabla de respuestas, que se pueden exportar en PDF y XLS.

Ninguna de las preguntas de texto depende de una configuración adicional, es decir, simplemente ingrese la pregunta que se mostrará al/los usuario/s. En el caso de preguntas de tipo Campo Formateado, es necesario configurar la máscara deseada.

Elementos de tipo numérico​

Los elementos de tipo número son aquellos donde los datos de entrada son numéricos o se pueden convertir a una cantidad. En este tipo de entrada de datos es posible realizar cálculos automáticos y también se generan los siguientes informes: análisis gráficos, extracto de colección y tabla de respuestas, que se pueden exportar en PDF y XLS.

Preguntas numéricas disponibles:​

Entero​

Permite el ingreso de números enteros (sin comas).

Ejemplo:

Decimal​

Permite el ingreso de un número racional (con coma).

Ejemplo:

emojis​

Permite el ingreso de emojis, que pueden representar la cantidad seleccionada.

Elementos de selección única​

Los elementos de selección única son aquellos en los que la entrada de datos permite seleccionar una de varias opciones (pregunta de opción múltiple con una sola respuesta). En este tipo de entrada de datos es posible realizar cálculos automáticos y también se generan los siguientes informes: análisis gráficos, extracto de colección y tabla de respuestas, que se pueden exportar en PDF y XLS.

Múltiples elementos de selección​

Los elementos de selección múltiple son aquellos en los que la entrada de datos permite seleccionar una o más opciones entre varias (pregunta de opción múltiple con más de una respuesta). En este tipo de entrada de datos es posible realizar cálculos automáticos y también se generan los siguientes informes: análisis gráficos, extracto de colección y tabla de respuestas, que se pueden exportar en PDF y XLS.

Elementos de evaluación​

Los elementos de evaluación son aquellos en los que el ingreso de datos permite seleccionar una entre varias opciones (pregunta de opción múltiple con una sola respuesta), pero con recursos gráficos (emojis). En este tipo de entrada de datos es posible realizar cálculos automáticos y también se generan los siguientes informes: análisis gráficos, extracto de colección y tabla de respuestas, que se pueden exportar en PDF y XLS.
