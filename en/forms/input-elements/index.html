Elements (Data Input)

The elements are the types of questions available for creating forms and are divided into two large groups, 1) Data entry and 2) Display.

Data entry elements are subdivided into: text, numbers, single selection, multiple selection, date and time, evaluation and others.

They are accessed from the form edit screen:

For detailed information, access the forms section.

Text type elements​

Text-type elements are those in which the data input is a string (text). In this type of data entry, the following reports are generated: collection extract and response table, which can be exported in PDF and XLS.

None of the text questions depend on additional configuration, that is, simply enter the question that will be displayed to the user/s. In the case of Formatted Field type questions, it is necessary to configure the desired mask.

Number type elements​

Elements of type number are those where the input data is numeric or can be converted to a quantity. In this type of data entry, it is possible to perform automatic calculations and the following reports are also generated: graphical analyses, collection extract and response table, which can be exported in PDF and XLS.

Available numerical questions:​

Whole​

Allows the entry of integers (without commas).

Decimal​

Allows the entry of a rational number (with comma).

Single select elements​

Single selection elements are those in which data entry allows you to select one of several options (multiple choice question with only one answer). In this type of data entry, it is possible to perform automatic calculations and the following reports are also generated: graphical analyses, collection extract and response table, which can be exported in PDF and XLS.
