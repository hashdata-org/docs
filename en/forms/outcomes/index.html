Resultados

Resultados é o local onde as informações das coletas são centralizadas, organizadas e disponibilizadas para análise. Este recurso foi projetado para fornecer uma visão abrangente dos dados coletados, permitindo que os usuários acompanhem o desempenho do formulário e tomem decisões informadas com base nos resultados.

Funcionalidades dos Resultados. Clique na imagem para ampliá-la.

O que você encontra em Resultados​

  1. Dados
  • Exibe uma lista detalhada com todas as respostas enviadas durante as coletas de dados;
  • Permite filtrar, ordenar e buscar informações específicas dentro das coletas realizadas;
  • Facilita o acompanhamento em tempo real, garantindo que as informações mais recentes estejam sempre acessíveis.
  • Permite exportação de dados nos formatos CSV ou no padrão Excel (.xlsx) ou geração de PDF configurável, para análises mais detalhadas.
  1. Gráficos
  • Representações visuais dos dados coletados, como gráficos de barras, pizza ou linhas, dependendo das configurações do formulário.
  • Auxiliam na identificação de tendências, padrões ou insights rapidamente.
  • Podem ser personalizados para destacar os dados mais relevantes para a análise.
  • Permite exportação dos gráficos no formato Word (.docx).
  1. Download de fotos
  • Armazena imagens enviadas pelos respondentes, como evidências visuais ou anexos relacionados às respostas.
  • Disponíveis para visualização direta na aba ou para download em lote.
  • Garantem que informações adicionais e contextos visuais estejam sempre acessíveis.
