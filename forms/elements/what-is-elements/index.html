Elementos

Os elementos são os componentes que constituem a estrutura do formulário, permitindo a coleta dos dados. Em outros termos, os elementos são os tipos de perguntas disponíveis para serem inseridos em um formulário.

Importante

As configurações realizadas nessa etapa definirão como o formulário será apresentado para a coleta de dados.

Apresentação das principais funcionalidades para inclusão e gerenciamento de elementos no formulário. Clique na imagem para ampliá-la.

1. Adicionando elementos ao formulário​

  1. Na barra lateral esquerda Elementos do formulário, clique em , disponível na porção inferior da barra lateral;
  2. Escolha o elemento desejado, os quais estão organizados em abas: Entrada de dados e Exibição;
  3. Clique no elemento desejado e ele será incluído no formulário.

1.1. Elementos de Entrada de Dados​

TipoElementosDescrição
Texto

  • Linha única: entrada de texto com inserção de várias linhas e parágrafos;

  • Várias linhas: entrada de texto com inserção de várias linhas e parágrafos;

  • Email: pré-formatado para endereços de correio eletrônico;

  • Máscara: entrada de texto personalizada, onde somente são aceitos valores especificados na máscara.

Usados para capturar respostas abertas, como nomes, endereços, comentários, e-mails, textos formatados (CPF, RG etc.).

Nesse tipo de entrada de dados são gerados os seguintes relatórios: extrato de coletas e tabela de respostas (que podem ser exportadas em PDF e XLS).

Nenhuma das questões do tipo texto dependem de configuração adicional, isto é, basta inserir a pergunta que será exibida no formulário.

No caso das questões do tipo Máscara, é necessário configurar a máscara desejada.

Por exemplo, para configurar um entrada de placa de veículo, pode-se utilizar uma máscara no formato AAA9999, que permite 3 letras maiúsculas, seguidas de 4 números. Saiba mais...

Número
  • Inteiro: entrada de números inteiros (sem vírgula);
  • Decimal: entrada de números racionais (com vírgula);
  • Quantidade (Emojis: entrada de dados com emojis, que representam quantidade selecionada.

Utilizados para entrada de dados numéricos ou que podem ser convertido em uma quantidade.

Nesse tipo de entrada de dados é possível realizar cálculos automáticos e também são gerados os seguintes relatórios: análises gráficas, extrato de coletas e tabela de respostas (que podem ser exportadas em PDF e XLS).

Seleção única
(várias opções, uma resposta)
  • Seleção única (Menu): as opções são mostradas com formato de um menu suspenso;
  • Seleção única (Check): as opções são mostradas com formato de caixas de seleção;
  • Seleção única (Botões): as opções são mostradas com formato de botões clicáveis.

Entrada de dados que permite selecionar uma dentre várias opções (questão de múltipla escolha com apenas uma resposta).

Nesse tipo de entrada de dados é possível realizar cálculos automáticos e também são gerados os seguintes relatórios: análises gráficas, extrato de coletas e tabela de respostas (que podem ser exportadas em PDF e XLS).

Múltipla seleção
(várias opções, várias respostas)
  • Multi-seleção (Check): as opções são mostradas com formato de caixas de seleção;
  • Multi-seleção (Botões): as opções são mostradas com formato de botões clicáveis.

Entrada de dados que permite selecionar uma ou mais opções dentre várias (questão de múltipla escolha com mais de uma resposta).

Nesse tipo de entrada de dados é possível realizar cálculos automáticos e também são gerados os seguintes relatórios: análises gráficas, extrato de coletas e tabela de respostas (que podem ser exportadas em PDF e XLS).

Data e hora
  • Data e hora: elemento que permite salvar data e horário;
  • Data: elemento que permite salvar somente data.
  • Hora: elemento que permite salvar somente o horário.
Utilizado como entrada de dados para salvar data, horário ou ambos por meio de um calendário.
Avaliação

  • Expressões faciais: elemento de múltipla seleção com emojis de expressões faciais configuráveis.

  • Emojis: elemento de múltipla seleção com emojis e quantidade questões configuráveis.

Entrada de dados que permite selecionar uma dentre várias opções (questão de múltipla escolha com apenas uma resposta), no entanto com recurso gráfico (emojis).

Nesse tipo de entrada de dados é possível realizar cálculos automáticos e também são gerados os seguintes relatórios: análises gráficas, extrato de coletas e tabela de respostas (que podem ser exportadas em PDF e XLS).

Outros
  • Enviar arquivos
  • GPS
  • Campo calculado (script)
  • Campo calculado (fórmula)
  • Ordenar / priorizar
  • Fotos / imagens
  • Assinatura

São elementos especiais, que possibilitam a criação de campos automatizados, com scripts e fórmulas, ou para carregar arquivos, fotografias e coordenadas de geolocalização.

1.2. Elementos de Exibição​

TipoElementoDescrição
Imagem
  • Exibição de imagem

Elemento utilizado para exibir uma imagem no formulário.

Para incluir uma imagem, você pode arrastar e soltar um arquivo ou clicar, para abrir uma caixa de diálogo para escolha do arquivo de imagem.

Texto
  • Exibição de texto

Elemento utilizado para exibir um texto no formulário.

Ao clicar em , será aberta um caixa de diálogo com um editor de texto para digitação e formatação do texto desejado.

2. Gerenciando os elementos inseridos​

Todos os elementos incluídos no formulário ficam disponíveis na barra lateral esquerda Elementos do formulário. Ao clicar em um elemento desses elementos, são mostradas um conjunto de opções para gerenciá-los. A partir dessas opções, você poderá:

  • Selecionar todos;
  • Mover selecionados;
  • Agrupar selecionados;
  • Clonar selecionados;
  • Excluir selecionados;

3. Configurando os elementos​

Os elementos adicionados ao formulário são mostrados na porção central da tela e, ao redor desses elementos, são apresentadas as seguintes opções de configuração:

OpçãoDescrição
Título, pergunta etc.

O texto principal do elemento pode ser uma pergunta ou simplesmente um título. Por exemplo, para coletar o nome completo, pode-se usar:

  • Nome completo:
Descrição

Embora seja opcional, a descrição do elemento pode ser utilizada para instruir a resposta. Por exemplo, em um campo para coleta do nome completo, pode-se usar a seguinte descrição:

  • Informe seu nome completo, sem abreviações.
Alterar o tipo da pergunta (elemento)

Permite a alteração de um elemento para outros elementos de tipos relacionados, após ter sido incluído no formulário.

Por exemplo, altere um elemento Linha única para o elemento Inteiro, do tipo Número. Ou um elemento Multi-seleção (Check) para Multi-seleção (Botões), ambos do tipo Múltipla seleção.

Ações condicionais

Ações condicionais são aplicadas automaticamente nos elementos do formulário quando determinadas condições acontecem. São blocos do tipo "SE/ENTÃO": SE (condição acontecer) ENTÃO (faça a ação).

Atenção

As ações condicionais são processadas SEMPRE que uma resposta do formulário é alterada.

Conteúdo condicional (multiconteúdo)

Permite criar vários conteúdos para a mesma pergunta, incluindo opções de respostas diversas, escolhendo qual conteúdo deve ser exibido de acordo com condições configuráveis. Por exemplo, imagine duas perguntas:

  • Qual o Estado?
  • Qual a Cidade?

Para facilitar a dinâmica ao responder o formulário, podemos exibir na listagem de cidades apenas as Cidades contidas no Estado selecionado. Para isso, basta habilitar o multiconteúdo (conteúdo condicional) na pergunta "Qual a Cidade?".

Assim, é possível criar uma lista de cidades para cada Estado e vincular a exibição de cada lista com a seleção do Estado.

A mesma lógica se aplica a vários outros cenários, por exemplo:

  • Setores, Áreas ↔️ Departamentos
  • Departamentos ↔️ Pessoas, Gerentes
  • Categorias ↔️ Produtos, Itens etc.
Opções de respostas

Opção específica para alguns elementos que requerem configurações adicionais, como aqueles que permitem Seleção única e Múltipla seleção.

Clonar

Gera cópias do elemento (pergunta). É possível escolher a quantidade de clones e a posição que devem ser inseridos no formulário.

MoverMove o elemento (pergunta) para uma determinada posição no formulário.
ExcluirExcluir o elemento (pergunta) do formulário.

3.1. Atributos do elemento e Atributos do conteúdo​

Ao configurar um elemento inserido no formulário, são mostradas, na barra lateral direita, as configurações dos atributos do elemento dos atributos do conteúdo. Veja, a seguir, cada uma dessas configurações:

3.1.1. Atributos do elemento​

AtributoDescrição Elementos disponíveis
Nome do campo

O nome do campo deve ser único (exclusivo) em todo o formulário e é utilizado na criação de fórmulas matemáticas do elemento do tipo campo calculado (script).

O nome do campo é usado também na exportação de dados para outros sistema via API, via arquivos CSV e XLSX, e na integração com o Hashdata Advanced Reports.

Por exemplo: peso_amostra_1, peso_amostra_2 e peso_amostra_3.

Todos os elementos do tipo Texto, Números, Seleção única, Múltiplas seleções, Data e hora, Avaliação e Outros.

Tags

As tags são usadas para agrupar questões e facilitar a criação de fórmulas matemáticas do elemento do tipo campo calculado (script).

Por exemplo, para calcular a média dos pesos de várias amostras, basta colocar uma tag em cada questão de peso - desta forma: #peso - e usar essa tag na fórmula: avgByTag('#peso'), onde avgByTag significa average by tag (média por marcador). Saiba mais...

Todos os elementos do tipo Texto, Números, Seleção única, Múltiplas seleções, Data e hora, Avaliação e Outros.

Permitir copiar resposta de coleta anterior

Com esta opção habilitada, é possível copiar (carregar) a resposta informada na coleta anterior.

Essa funcionalidade é muito útil quando existem respostas que se repetem para um conjunto de coletas, por exemplo: Data, Cidade, Estado etc.

Todos os elementos do tipo Texto, Números, Seleção única, Múltiplas seleções, Data e hora, Avaliação e nos elementos GPS e Ordenar / priorizar do tipo Outros.

3.1.2. Atributos do conteúdo​

AtributoDescrição Elementos disponíveis
Preenchimento obrigatório

Indica se o elemento deve de ser ou não obrigatoriamente preenchido antes que o formulário possa ser enviado ou processado.

Quando essa opção está ativada, o usuário não poderá prosseguir ou concluir a submissão do formulário sem inserir uma resposta válida no campo especificado.

Todos os elementos do tipo Texto, Números, Seleção única, Múltiplas seleções, Data e hora, Avaliação e elementos Enviar arquivos, GPS, Campo calculado (script), Campo calculado (fórmula), Ordenar / priorizar, Fotos / imagens, Assinatura do tipo Outros.

Permitir inserir comentário

Habilita um campo adicional ao elemento, onde é permitido escrever livremente um comentário ou observação sobre a questão.

O comentário pode ser útil em perguntas que requerem ou necessitam de explicações adicionais, feedback ou detalhes que não se encaixam nas respostas fechadas.

Ao habilitar essa opção, é mostrado o atributo Comentário obrigatório, para indicar a obrigatoriedade do preenchimento do comentário.

Todos os elementos do tipo Texto, Números, Seleção única, Múltiplas seleções, Data e hora, Avaliação e elementos Enviar arquivos, GPS, Campo calculado (script), Campo calculado (fórmula), Ordenar / priorizar, Fotos / imagens, Assinatura do tipo Outros.

Permitir inserir fotos

Este atributo indica se é ou não permitido carregar fotos ao elemento. Ao habilitar essa opção, são mostrados os seguintes atributos. Saiba mais...

  • Permitir fotos da galeria;
  • Mínimo de fotos requeridas;
  • Máximo de fotos permitidas;
  • Tamanho máximo das fotos.

Todos os elementos do tipo Texto, Números, Seleção única, Múltiplas seleções, Data e hora, Avaliação e elementos GPS, Campo calculado (script), Campo calculado (fórmula), Ordenar / priorizar do tipo Outros.

Mínimo de caracteres requerido

Define a quantidade mínima de caracteres que o usuário deve inserir no elemento para que a resposta seja válida.

Ao configurar essa opção, o formulário garante que as respostas fornecidas pelos usuários tenham um tamanho mínimo, evitando respostas curtas demais ou incompletas.

Para configurar, escolha valores entre 0 e 4.999.

Elementos Linha única e Várias linhas do tipo Texto.

Máximo de caracteres permitido

Restringe o número máximo de caracteres que pode ser inserido em um elemento de entrada de dados para que a resposta seja considerada válida.

Esse recurso é fundamental para garantir que os dados sejam formatados corretamente e para evitar sobrecarga de informações.

Para configurar, escolha valores entre 1 e 5.000.

Elementos Linha única e Várias linhas do tipo Texto.

Contagem de linhas iniciais

Escolha valores entre 1 e 20.

Elemento Várias linhas do tipo Texto.

Máscara

Define um formato específico para o elemento de entrada de dados. A máscara é utilizada para guiar o usuário sobre como deve ser o formato das informações que estão sendo inseridas, garantindo que os dados sejam coletados de forma consistente e correta.

Estão pré-configuradas as seguintes máscaras:

  • CPF - 999.999.999-99
  • CNPJ - 99.999.999/9999-99
  • CEP - 99.999-999
  • Telefone - (99) 99999-9999
  • Placa veicular - AAA-9999/AA

  • Número de 3 dígitos com sinal e duas casas decimais opcionais. Aceita virgula e ponto c+omo separador decimal. - [~]999[(,|.)99]

O campo é editável e permite a inclusão de máscaras personalizadas.

Máscaras personalizadas

Para configurar máscaras personalizadas, podem ser usados os seguintes caracteres especiais:

  • 9: número
  • a: letra minúscula (a...z)
  • A: letra maiúscula (A...Z)
  • x: letra (minúscula ou maiúscula)
  • *: letra ou número
  • ~: sinal de positivo ou negativo (+ ou -)
  • (): bloco de grupo
  • {}: bloco de repetição
  • []: bloco opcional
  • \: cancela o efeito coringa do próximo caractere especial (9aAx*~)

É possível mesclar os caracteres especiais com outros caracteres, exemplo:

  • Número de série: PWR-999-aa9aa
Elemento Máscara do tipo Texto.
Mostrar separador de milhar

Define se o separador de milhar do elemento deve ser adicionado automaticamente ao elemento. Este recurso facilita a leitura e a compreensão de números grandes, organizando-os em grupos de três dígitos.

À medida que o usuário digita um número, o separador de milhar é inserido automaticamente no local apropriado. Por exemplo, ao digitar "1000000", o número é formatado como "1.000.000" ou "1,000,000", dependendo do formato regional.

Elemento Inteiro do tipo Números.
Número mínimo

Determina o valor mais baixo que pode ser inserido no elemento. Ao limitar a faixa de valores que um usuário pode inserir em um formulário, garante que os dados respeitem um limite inferior previamente estabelecido.

Elementos Inteiro e Decimal do tipo Números.
Número máximo

Determina o valor mais alto que pode ser inserido no elemento. Ao limitar a faixa de valores que um usuário pode inserir em um formulário, garante que os dados respeitem um limite superior previamente estabelecido.

Elementos Inteiro e Decimal do tipo Números.
Prefixo

Prefixo é um texto que precede o valor númerico a ser digitado pelo usuário, ajudando-o a entender que tipo de informação ele precisa digitar.

Por exemplo, em campo de valor monetário, o prefixo poderia ser: "R$ " ou "USD".

Recomendação

Coloque um espaço depois do texto do prefixo, para que este não fique junto (colado) com valor informado.

Elementos Inteiro e Decimal do tipo Números e elementos Campo calculado (script) e Campo calculado (fórmula) do tipo Outros.

Sufixo

Sufixo é um texto que sucede o valor númerico a ser digitado pelo usuário, ajudando-o a entender que tipo de informação ele precisa digitar.

Por exemplo, em campo de unidades de medidas ou distâncias, os sufixos poderiam ser: "kg" e "km ".

Recomendação

Coloque um espaço antes do texto do sufixo, para que este não fique junto (colado) com valor informado.

Elementos Inteiro e Decimal do tipo Números e elementos Campo calculado (script) e Campo calculado (fórmula) do tipo Outros.

Casas Decimais

Define a quantidade de dígitos permitidos após o ponto decimal no elemento numérico.

Esse recurso é útil para controlar a precisão dos valores inseridos, como em cálculos financeiros, medidas ou outras entradas que exijam decimais. Escolha entre 0 e 8 casas decimais.

Elemento Decimal do tipo Números e elementos Campo calculado (script) e Campo calculado (fórmula) do tipo Outros.

Mostrar números

Define se os caracteres numéricos referentes às quantidades devem ou não mostradas juntamente com os emojis.

Elemento Quantidade (Emojis) do tipo Números e elemento Emojis do tipo Avaliação.
Tipo

Permite a alteração do elemento dentre outros do mesmo tipo.

Por exemplo: possibilita trocar o elemento Seleção única (Menu) por Seleção única (Botões) ou Seleção única (Check), todos do tipo Seleção única (várias opções, uma resposta).

Todos os elementos dos tipos Seleção única e Múltiplas seleções.
Número de colunas

Esse atributo define a forma de exibição das opções, mais especificamente, quantas opções serão exibidas por linha:

  • Se o número de colunas for 1, então todas as opções serão exibidas uma embaixo da outra (uma por linha);

  • Se o número de coletas for 2, então serão exibidas duas opções de resposta por linha (2 colunas).

Todos os elementos dos tipos Seleção única e Múltiplas seleções.
Seleção mínima

Define a quantidade mínima de itens que um usuário deve selecionar no elemento de múltipla seleção para que a resposta seja considerada válida.

Antes de indicar o valor deste atributo, é necessário configurar em Opções de respostas, a quantidade de itens que compõe o elemento.

Todos os elementos do tipo Múltiplas seleções.
Seleção máxima

Define a quantidade máxima de itens que um usuário deve selecionar no elemento de múltipla seleção para que a resposta seja considerada válida.

Antes de indicar o valor deste atributo, é necessário configurar em Opções de respostas, a quantidade de itens que compõe o elemento.

Todos os elementos do tipo Múltiplas seleções.
Data mínima

Definir o primeiro dia disponível para seleção da data no elemento. Ao configurar essa opção, o sistema estabelece um limite inferior, impedindo a seleção de datas anteriores à data mínima configurada.

Elementos Data e hora e Data do tipo Data e hora.
Data máxima

Define um limite máximo de data que pose ser escolhida para o elemento. Ao configurar uma data máxima, o sistema restringe a seleção de datas a um período que não ultrapasse essa data limite.

Elementos Data e hora e Data do tipo Data e hora.
Mostrar calendário

Define se o calendário dever ser ou não mostrado durante a escolha de uma data no preenchimento do formulário.

Ao habilitar essa opção, será exibido um calendário, geralmente mensal, para que o usuário possa consultar datas, programar eventos ou selecionar uma data específica.

Elementos Data e hora e Data do tipo Data e hora.
Formato

Define o sistema de exibição da hora. Escolha entre 12 e 24.

O formato de 12 horas divide o dia em dois períodos de 12 horas: a parte da manhã (AM - ante meridiem) e a parte da tarde/noite (PM - post meridiem). Por exemplo, 2:00 PM indica 14 horas, ou seja, duas horas da tarde.

No formato de 24 horas, as horas são contadas continuamente de 00:00 a 23:59, sem a necessidade de usar AM ou PM.

Elementos Data e hora e Hora do tipo Data e hora.
Mostrar Segundos

Define os segundos dever ser ou não mostrado durante a escolha de um horário no preenchimento do formulário.

Elementos Data e hora e Hora do tipo Data e hora.
Quantidade mínima de arquivos

Define a quantidade mínima de arquivos que um usuário deve carregar no elemento para que a resposta seja considerada válida.

Elemento Enviar arquivos do tipo Outros.
Quantidade máxima de arquivos

Define a quantidade máxima de arquivos que um usuário deve carregar no elemento para que a resposta seja considerada válida.

Elemento Enviar arquivos do tipo Outros.
Permitir qualquer tipo de arquivo

Ao habilitar esta opção, qualquer tipo de arquivo pode ser carregada no formulário.

Ao desabilitar, são mostradas opções para que se defina quais tipos de arquivos podem ser carregados:

  • Permitir áudios
  • Permitir videos
  • Permitir imagens
  • Permitir documentos
Elemento Enviar arquivos do tipo Outros.
Permitir digitar as coordenadas manualmente

Define se as coordenadas geográficas devem ser obtidas somente pelo sensor de GPS ou podem ser digitadas durante a coleta de dados.

Elemento GPS do tipo Outros.
Precisão mínima requerida (m)

Define o nível de exatidão necessário para uma captura de localização GPS do elemento. Com isso, sistema somente aceitará a coordenada GPS se ela atingir ou exceder o nível de precisão especificado.

A precisão é medida em metros e valores mais baixos indicam maior exatidão. Portanto, escolha valores entre 100 m e 10.000 m, em intervalos de 50 m.

Elemento GPS do tipo Outros.
Ocultar resultado do usuário

Define ou não se os resultados cálculos realizados no elemento serão mostrados durante a coleta de dados.

Elementos Campo calculado (script) e Campo calculado (fórmula) do tipo Outros.
Permitir fotos da galeria

Indica se permite ou não carregar fotos ao formulário a partir da galeria do dispositivo.

Caso não seja permitido, será possível carregar fotos apenas a partir da câmera durante a coleta de dados.

Elemento Fotos / imagens do tipo Outros e todos os elementos onde há o atributo Permitir inserir fotos.

Mínimo de fotos requeridas

Indica a quantidade mínima de fotos que podem ser carregadas no formulário. Escolha valores entre 0 e 100.

Elemento Fotos / imagens do tipo Outros e todos os elementos onde há o atributo Permitir inserir fotos.

Máximo de fotos permitidas

Indica a quantidade máxima de fotos que podem ser carregadas no formulário. Escolha valores entre 1 e 100.

Elemento Fotos / imagens do tipo Outros e todos os elementos onde há o atributo Permitir inserir fotos.

Tamanho máximo das fotos

Define a dimensão máxima, em pixels, de cada fotografia carregada ao formulário. Escolha uma dimensão máxima dentre as seguintes opções:

  • 512 x 512
  • 768 x 768
  • 1024 x 1024
  • 1536 x 1536
  • 2048 x 2048

Elemento Fotos / imagens do tipo Outros e todos os elementos onde há o atributo Permitir inserir fotos.

Máxima largura da linha

Define a espessura da linha da assinatura. Escolha valores entre 1 e 10.

Elemento Assinatura do tipo Outros.

Raio de um único ponto

Escolha valores entre 1 a 5.

Elemento Assinatura do tipo Outros.

Cor da linha

Define a cor da linha da assinatura. Ao clicar no menu suspenso, é mostrada uma paleta de cores disponíveis.

Elemento Assinatura do tipo Outros.

Alinhamento

Define o alinhamento da imagem no formulário, se deve ser à esquerda, à direita ou ao centro. Escolha entre: esquerda, direita e centro.

Elemento Exibição de imagem do tipo Imagem, pertencente aos Elementos de Exibição.

Arredondamento da imagem

Define o arredondamento dos cantos da imagem exibida no formulário. Escolha valores entre 0 e 300.

Elemento Exibição de imagem do tipo Imagem, pertencente aos Elementos de Exibição.

Largura máxima

Define a largura máxima, em pixels, da imagem exibida no formulário. Escolha valores entre 50 e 1024.

Elemento Exibição de imagem do tipo Imagem, pertencente aos Elementos de Exibição.

Sombra

Indica se exibe ou não sombreamento na imagem exibida no formulário.

Elemento Exibição de imagem do tipo Imagem, pertencente aos Elementos de Exibição.

Exibir título ao usuário

Indica se o título do elemento deve ser exibido ou não no formulário durante a coleta de dados.

Elemento Exibição de texto do tipo Texto, pertencente aos Elementos de Exibição.
